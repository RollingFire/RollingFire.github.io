Pizza Shop Microservice Application

Spring Boot Microservice application with REST APIs, an Angular Frontend, and a MySQL database of a Pizza Shop or any other restaurant. Provides menus, inventory management, order tracking, customer data, automatic inventory reorder, and a recipe book.

I started the project as a means to reinforce my learnings of Spring Boot and Angular. I had thought about opening a pizza shop at one point in my life and was wanting pizza that day, so naturally I decided to base the project on a pizza shop. Once I started working on it, I quickly realized that this could work of any restaurant and thus I learned that an entire application could follow object-oriented design principles and not only code.

This project was also my first project (outside of a classroom setting) that I utilized automated integration and unit tests. I quickly saw the value in it as it help me more quickly catch errors and giving me the confidence that all of the microservices are continue to work well together if I make a small change in one section of code. Also by writing the unit tests first, it helped me to stay organized in my thoughts and keep separate design tasks and development tasks.
